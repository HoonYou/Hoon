## VS Code 화면 전환 핵심 단축키

### 코드 ↔ 터미널
**Ctrl + fn + `**

- 터미널 열기 / 닫기
- 이미 열려 있으면 터미널로 포커스 이동
- 다시 누르면 에디터로 복귀

---

### 코드 ↔ 폴더(Explorer)
**Ctrl + Shift + E**

- 왼쪽 폴더 트리로 포커스 이동
- 파일 선택 후 Enter → 코드 열기

---

### 폴더 / 터미널 → 코드 화면
**Ctrl + 1**

- 첫 번째 에디터 그룹(코드 영역)으로 이동

---

### 사이드바 숨기기 / 표시
**Ctrl + B**

- 왼쪽 사이드바 on/off
- 코드 집중 모드에 유용
